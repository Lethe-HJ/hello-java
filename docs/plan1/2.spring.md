# Spring Boot 接口极简入门

目标：掌握 Spring Boot 最常用的接口写法，能启动项目、写 GET / POST 接口、接收前端参数，并按 `controller -> service` 做简单分层。

---

## 1. 创建 Spring Boot 项目

使用 Spring Initializr 创建项目：

<https://start.spring.io/>

推荐选择：

| 配置 | 选择 |
| --- | --- |
| Project | Maven |
| Language | Java |
| Spring Boot | 默认稳定版本 |
| Packaging | Jar |
| Java | 8 或 17，按本地环境选择 |

依赖先只选一个：

* Spring Web

---

## 2. 项目目录结构

常见结构如下：

```text
src/main/java/com/example/demo
├── DemoApplication.java
├── controller
│   └── HelloController.java
├── service
│   └── UserService.java
└── model
    └── User.java
```

先记住：

* `controller`：接收前端请求，返回数据
* `service`：写业务逻辑
* `model`：定义数据对象

---

## 3. 启动项目

Spring Boot 项目入口通常长这样：

```java
package com.example.demo;

import org.springframework.boot.SpringApplication;
import org.springframework.boot.autoconfigure.SpringBootApplication;

@SpringBootApplication
public class DemoApplication {

    public static void main(String[] args) {
        SpringApplication.run(DemoApplication.class, args);
    }
}
```

启动后，默认访问地址：

```text
http://localhost:8080
```

---

## 4. 写第一个 GET 接口

创建 `controller/HelloController.java`：

```java
package com.example.demo.controller;

import java.util.HashMap;
import java.util.Map;

import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.RestController;

@RestController
public class HelloController {

    @GetMapping("/hello")
    public Map<String, Object> hello() {
        Map<String, Object> result = new HashMap<>();
        result.put("msg", "hello world");
        return result;
    }
}
```

访问：

```text
GET http://localhost:8080/hello
```

返回：

```json
{
  "msg": "hello world"
}
```

## 5. 必学注解

### `@RestController`

表示这个类专门用来写接口。

返回 `String`、对象、`Map`、`List` 时，Spring Boot 会自动转成响应内容。

```java
@RestController
public class HelloController {
}
```

### `@GetMapping`

表示接收 GET 请求，常用于查询数据。

```java
@GetMapping("/hello")
public String hello() {
    return "hello";
}
```

### `@PostMapping`

表示接收 POST 请求，常用于新增数据或提交表单。

```java
@PostMapping("/user")
public User createUser(@RequestBody User user) {
    return user;
}
```

### `@RequestParam`

接收 Query 参数。

```java
@GetMapping("/user")
public String getUser(@RequestParam Long id) {
    return "user id = " + id;
}
```

请求：

```text
GET http://localhost:8080/user?id=1
```

### `@RequestBody`

接收 JSON 请求体。

```java
@PostMapping("/user")
public User createUser(@RequestBody User user) {
    return user;
}
```

请求体：

```json
{
  "id": 1,
  "name": "Tom",
  "age": 18
}
```

## 6. 定义 User 对象

创建 `model/User.java`：

```java
package com.example.demo.model;

public class User {

    private Long id;
    private String name;
    private Integer age;

    public Long getId() {
        return id;
    }

    public void setId(Long id) {
        this.id = id;
    }

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }

    public Integer getAge() {
        return age;
    }

    public void setAge(Integer age) {
        this.age = age;
    }
}
```

## 7. GET 接收 Query 参数

创建 `controller/UserController.java`：

```java
package com.example.demo.controller;

import com.example.demo.model.User;
import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.RequestParam;
import org.springframework.web.bind.annotation.RestController;

@RestController
public class UserController {

    @GetMapping("/user")
    public User getUser(@RequestParam Long id) {
        User user = new User();
        user.setId(id);
        user.setName("Tom");
        user.setAge(18);
        return user;
    }
}
```

访问：

```text
GET http://localhost:8080/user?id=1
```

返回：

```json
{
  "id": 1,
  "name": "Tom",
  "age": 18
}
```

## 8. POST 接收 JSON

在 `UserController` 中增加：

```java
import org.springframework.web.bind.annotation.PostMapping;
import org.springframework.web.bind.annotation.RequestBody;
```

```java
@PostMapping("/user")
public User createUser(@RequestBody User user) {
    return user;
}
```

请求：

```text
POST http://localhost:8080/user
Content-Type: application/json
```

请求体：

```json
{
  "id": 2,
  "name": "Jerry",
  "age": 20
}
```

返回：

```json
{
  "id": 2,
  "name": "Jerry",
  "age": 20
}
```

## 9. 简单分层：controller -> service

不要把所有逻辑都写在 `controller` 里。

先记住这个结构：

```text
前端请求 -> controller -> service -> 返回结果
```

创建 `service/UserService.java`：

```java
package com.example.demo.service;

import com.example.demo.model.User;
import org.springframework.stereotype.Service;

@Service
public class UserService {

    public User getUser(Long id) {
        User user = new User();
        user.setId(id);
        user.setName("Tom");
        user.setAge(18);
        return user;
    }

    public User createUser(User user) {
        return user;
    }
}
```

修改 `UserController.java`：

```java
package com.example.demo.controller;

import com.example.demo.model.User;
import com.example.demo.service.UserService;
import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.PostMapping;
import org.springframework.web.bind.annotation.RequestBody;
import org.springframework.web.bind.annotation.RequestParam;
import org.springframework.web.bind.annotation.RestController;

@RestController
public class UserController {

    private final UserService userService;

    public UserController(UserService userService) {
        this.userService = userService;
    }

    @GetMapping("/user")
    public User getUser(@RequestParam Long id) {
        return userService.getUser(id);
    }

    @PostMapping("/user")
    public User createUser(@RequestBody User user) {
        return userService.createUser(user);
    }
}
```

---

## 10. 当前阶段只需要记住

| 注解 | 作用 |
| --- | --- |
| `@RestController` | 声明接口类 |
| `@GetMapping` | 写 GET 接口 |
| `@PostMapping` | 写 POST 接口 |
| `@RequestParam` | 接收 Query 参数 |
| `@RequestBody` | 接收 JSON 请求体 |
| `@Service` | 声明业务类 |
