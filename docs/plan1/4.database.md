# MySQL + MyBatis CRUD 极简入门

目标：掌握最常用的 MySQL CRUD，能让 Spring Boot 通过 MyBatis 连接数据库，并完成用户的查询、新增、修改、删除接口。

---

## 1. 这几个部分分别做什么

```text
前端请求 -> controller -> service -> mapper -> MySQL
```

* `controller`：接收 HTTP 请求，返回 JSON
* `service`：写业务逻辑、调用 `mapper`
* `mapper`：Java 中执行 SQL 的接口
* MySQL：真正保存数据的数据库

本篇继续使用前一篇中的 `User`、`UserController`、`UserService`。

---

## 2. 安装并启动 MySQL

如果电脑还没有 MySQL，Mac 可以用 Homebrew 安装：

```bash
brew install mysql
brew services start mysql
```

确认 MySQL 已启动：

```bash
mysql --version
mysql -u root -p
```

第二条命令会要求输入 root 用户的密码。能进入下面的命令行就表示连接成功：

```text
mysql>
```

Windows 可以安装 MySQL Installer；安装时记住 root 密码即可。

## 3. 创建数据库和用户表

在 MySQL 命令行中执行：

```sql
CREATE DATABASE demo DEFAULT CHARACTER SET utf8mb4;

USE demo;

CREATE TABLE users (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    name VARCHAR(50) NOT NULL,
    age INT
);
```

说明：

| 内容          | 含义                                                |
| ------------- | --------------------------------------------------- |
| `demo`        | 数据库名称                                          |
| `users`       | 表名称。用复数可避免和 MySQL 的 `USER` 相关名称混淆 |
| `id`          | 用户 ID；`AUTO_INCREMENT` 表示新增时自动生成        |
| `VARCHAR(50)` | 最多 50 个字符的文本                                |
| `NOT NULL`    | 这个字段不能为空                                    |

插入两条测试数据：

```sql
INSERT INTO users (name, age) VALUES ('Tom', 18);
INSERT INTO users (name, age) VALUES ('Jerry', 20);
```

查看表中的数据：

```sql
SELECT * FROM users;
```

## 4. 必学 SQL：CRUD

CRUD 就是新增（Create）、查询（Read）、修改（Update）、删除（Delete）。

### 4.1 查询：SELECT

查询全部用户：

```sql
SELECT id, name, age FROM users;
```

按 ID 查询一个用户：

```sql
SELECT id, name, age FROM users WHERE id = 1;
```

### 4.2 新增：INSERT

```sql
INSERT INTO users (name, age) VALUES ('Alice', 22);
```

`id` 会由数据库自动生成，所以不需要写。

### 4.3 修改：UPDATE

```sql
UPDATE users
SET name = 'Alice Chen', age = 23
WHERE id = 3;
```

注意：`UPDATE` 一定要写 `WHERE`。没有 `WHERE` 会修改整张表的所有数据。

### 4.4 删除：DELETE

```sql
DELETE FROM users WHERE id = 3;
```

同样，`DELETE` 一定要写 `WHERE`。没有 `WHERE` 会删除整张表的数据。

## 5. 引入 MyBatis 和 MySQL 驱动

打开项目根目录的 `pom.xml`，在 `<dependencies>` 中加入：

```xml
<dependency>
    <groupId>org.mybatis.spring.boot</groupId>
    <artifactId>mybatis-spring-boot-starter</artifactId>
    <version>3.0.4</version>
</dependency>

<dependency>
    <groupId>com.mysql</groupId>
    <artifactId>mysql-connector-j</artifactId>
    <scope>runtime</scope>
</dependency>
```

`mybatis-spring-boot-starter` 让 Spring Boot 能使用 MyBatis；`mysql-connector-j` 是 Java 连接 MySQL 所需的驱动。

加入依赖后，执行：

```bash
./mvnw clean package
```

Windows 使用：

```bash
mvnw.cmd clean package
```

## 6. 配置数据库连接

编辑 `src/main/resources/application.properties`：

```properties
spring.datasource.url=jdbc:mysql://localhost:3306/demo?useUnicode=true&characterEncoding=utf8&serverTimezone=Asia/Shangha
spring.datasource.username=root
spring.datasource.password=你的MySQL密码
spring.datasource.driver-class-name=com.mysql.cj.jdbc.Driver
```

把 `你的MySQL密码` 换成自己安装 MySQL 时设置的 root 密码。这个文件不要提交包含真实密码的版本到公开仓库。

配置内容：

| 配置                | 含义                                                    |
| ------------------- | ------------------------------------------------------- |
| `url`               | 数据库地址；`3306` 是 MySQL 默认端口，`demo` 是数据库名 |
| `username`          | MySQL 用户名                                            |
| `password`          | MySQL 密码                                              |
| `driver-class-name` | MySQL 的 Java 驱动                                      |

## 7. 创建 Mapper

创建 `mapper/UserMapper.java`：

```java
package com.example.demo.mapper;

import com.example.demo.model.User;
import org.apache.ibatis.annotations.Delete;
import org.apache.ibatis.annotations.Insert;
import org.apache.ibatis.annotations.Mapper;
import org.apache.ibatis.annotations.Options;
import org.apache.ibatis.annotations.Param;
import org.apache.ibatis.annotations.Select;
import org.apache.ibatis.annotations.Update;

import java.util.List;

@Mapper
public interface UserMapper {

    @Select("SELECT id, name, age FROM users")
    List<User> findAll();

    @Select("SELECT id, name, age FROM users WHERE id = #{id}")
    User findById(@Param("id") Long id);

    @Insert("INSERT INTO users (name, age) VALUES (#{name}, #{age})")
    @Options(useGeneratedKeys = true, keyProperty = "id")
    int insert(User user);

    @Update("UPDATE users SET name = #{name}, age = #{age} WHERE id = #{id}")
    int update(User user);

    @Delete("DELETE FROM users WHERE id = #{id}")
    int deleteById(@Param("id") Long id);
}
```

重点：

* `@Mapper`：告诉 Spring 这是一个 MyBatis Mapper。
* `#{id}`、`#{name}`：把 Java 参数安全地传入 SQL；不要用字符串拼接 SQL。
* `@Options(...)`：新增后，把数据库生成的 ID 写回 `user.id`。
* `int` 返回受影响的行数；成功修改或删除一条数据时通常为 `1`。

## 8. 修改 Service

把 `service/UserService.java` 改为：

```java
package com.example.demo.service;

import com.example.demo.mapper.UserMapper;
import com.example.demo.model.User;
import org.springframework.stereotype.Service;

import java.util.List;

@Service
public class UserService {

    private final UserMapper userMapper;

    public UserService(UserMapper userMapper) {
        this.userMapper = userMapper;
    }

    public List<User> findAll() {
        return userMapper.findAll();
    }

    public User findById(Long id) {
        return userMapper.findById(id);
    }

    public User create(User user) {
        userMapper.insert(user);
        return user;
    }

    public boolean update(User user) {
        return userMapper.update(user) > 0;
    }

    public boolean deleteById(Long id) {
        return userMapper.deleteById(id) > 0;
    }
}
```

现在 `service` 不再创建假数据，而是通过 `mapper` 读写 MySQL。

## 9. 完成完整 CRUD 接口

把 `controller/UserController.java` 改为：

```java
package com.example.demo.controller;

import com.example.demo.model.User;
import com.example.demo.service.UserService;
import org.springframework.web.bind.annotation.DeleteMapping;
import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.PathVariable;
import org.springframework.web.bind.annotation.PostMapping;
import org.springframework.web.bind.annotation.PutMapping;
import org.springframework.web.bind.annotation.RequestBody;
import org.springframework.web.bind.annotation.RestController;

import java.util.List;

@RestController
public class UserController {

    private final UserService userService;

    public UserController(UserService userService) {
        this.userService = userService;
    }

    @GetMapping("/users")
    public List<User> findAll() {
        return userService.findAll();
    }

    @GetMapping("/user/{id}")
    public User findById(@PathVariable Long id) {
        return userService.findById(id);
    }

    @PostMapping("/user")
    public User create(@RequestBody User user) {
        return userService.create(user);
    }

    @PutMapping("/user")
    public boolean update(@RequestBody User user) {
        return userService.update(user);
    }

    @DeleteMapping("/user/{id}")
    public boolean deleteById(@PathVariable Long id) {
        return userService.deleteById(id);
    }
}
```

新注解：

| 注解             | 作用                                           |
| ---------------- | ---------------------------------------------- |
| `@PutMapping`    | 接收 PUT 请求，通常用于修改                    |
| `@DeleteMapping` | 接收 DELETE 请求，通常用于删除                 |
| `@PathVariable`  | 接收 URL 路径中的参数，例如 `/user/1` 中的 `1` |

## 10. 测试接口

启动项目：

```bash
./mvnw spring-boot:run
```

### 查询全部用户

```text
GET http://localhost:8080/users
```

### 查询单个用户

```text
GET http://localhost:8080/user/1
```

### 新增用户

```text
POST http://localhost:8080/user
Content-Type: application/json
```

```json
{
  "name": "Alice",
  "age": 22
}
```

返回中会带上数据库自动生成的 `id`：

```json
{
  "id": 3,
  "name": "Alice",
  "age": 22
}
```

### 修改用户

```text
PUT http://localhost:8080/user
Content-Type: application/json
```

```json
{
  "id": 3,
  "name": "Alice Chen",
  "age": 23
}
```

返回 `true` 表示修改成功。

### 删除用户

```text
DELETE http://localhost:8080/user/3
```

返回 `true` 表示删除成功。
