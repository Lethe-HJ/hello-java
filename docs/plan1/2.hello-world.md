# Hello World 实操指南

## 1. Windows 通过 WSL 安装环境

### 1.1 安装 WSL

WSL (Windows Subsystem for Linux) 是 Windows 内置的 Linux 子系统，可以在 Windows 上运行 Linux 环境。无论做 Web 前端还是 Web 后端，都建议使用 WSL 作为 Windows 系统的开发环境，原因如下：

1. **生态匹配**：Linux 是 Java、Node.js 等主流开发技术栈的原生运行平台，拥有更完善的命令行工具链和开发环境支持。
2. **学习成本**：WSL 提供的 Linux 环境与主流教程、文档及 AI 训练语料中的操作方式高度一致，能有效降低环境配置和学习成本。
3. **环境隔离**：WSL 本身是一个隔离环境，即使 AI 生成的命令或脚本出现问题，也不会影响到 Windows 主机环境。
4. **提前学习**：可以更早接触 Linux 系统，获得学习 Linux 系统的机会。
5. **环境一致**：开发环境与生产环境更加一致，减少部署时的环境差异问题。

**前提条件**：需要 Windows 10 2004 版本及以上，或 Windows 11。

以管理员身份打开 PowerShell，执行：

```powershell
wsl --install
```

这会自动启用 WSL 功能并安装 Ubuntu 发行版。安装完成后，重启电脑。

重启后，打开 Ubuntu 终端，设置用户名和密码。

验证 WSL 是否安装成功：

```bash
cat /etc/os-release
```

### 1.2 在 WSL 中安装 OpenJDK

在 WSL 的 Ubuntu 终端中，更新软件包列表并安装 OpenJDK 8：

```bash
sudo apt update
sudo apt install openjdk-8-jdk
```

安装完成后，验证 Java 是否安装成功：

```bash
java -version
javac -version
```

如果显示版本号，说明安装成功。

查看 Java 安装路径：

```bash
which java
which javac
```

### 1.3 安装 VS Code 和 Remote - WSL 插件

首先安装 VS Code，然后安装 Remote - WSL 插件，这样可以在 VS Code 中直接访问 WSL 中的文件。

**第一步**：在本地 Windows 的 VS Code 中安装 Remote - WSL 插件：

```text
Remote - WSL
```

**第二步**：点击 VS Code 左下角的绿色图标：

```text
><
```

选择：

```text
New WSL Window
```

这样就打开了 WSL 环境的 VS Code。

**第三步**：在 WSL Remote 窗口中安装 Java 插件：

```text
Extension Pack for Java
```

注意：Java 插件需要安装在 WSL 侧，而不是本地 Windows 侧。

### 1.4 在 WSL 中创建项目目录

在 WSL 终端中创建项目目录：

```bash
mkdir -p ~/workspace/hello-java
cd ~/workspace/hello-java
```

## 2. 写第一个 Java 文件

### 2.1 创建 Hello.java

在 WSL 窗口的 VS Code 中，打开刚才创建的目录：

```text
File -> Open Folder -> /home/你的用户名/workspace/hello-java
```

新建文件：

```text
Hello.java
```

写入代码：

```java
public class Hello {
    public static void main(String[] args) {
        System.out.println("Hello Java");
    }
}
```

注意：文件名 `Hello.java` 要和 `public class Hello` 的类名一致。

### 2.2 在 VS Code 里运行 Java

打开 `Hello.java` 后，通常可以直接点击代码上方的：

```text
Run
```

也可以右键代码文件，选择：

```text
Run Java
```

输出窗口中看到：

```text
Hello Java
```

说明运行成功。

### 2.3 用 WSL 命令行编译和运行

在 WSL 终端中，进入项目目录：

```bash
cd ~/workspace/hello-java
```

编译：

```bash
javac Hello.java
```

编译成功后，会生成：

```text
Hello.class
```

运行：

```bash
java Hello
```

注意：运行时写 `java Hello`，不要写 `java Hello.class`。

完整流程：

```bash
javac Hello.java
java Hello
```
