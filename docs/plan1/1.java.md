# Java 8 极简入门

目标：掌握 Java 8 最常用的语法，能读懂并写出简单代码。

---

## 1. 基本类型和变量

Java 变量必须声明类型。

```java
String name = "Tom";
int age = 18;
long id = 1001L;
double score = 99.5;
boolean enabled = true;
```

常用类型：

| 类型 | 含义 | 示例 |
| --- | --- | --- |
| `String` | 字符串 | `"Tom"` |
| `int` | 整数 | `18` |
| `long` / `Long` | 较大的整数 | `1001L` |
| `double` | 小数 | `99.5` |
| `boolean` | 布尔值 | `true` / `false` |

---

## 2. 条件和循环

```java
if (age >= 18) {
    System.out.println("adult");
} else {
    System.out.println("child");
}
```

```java
for (int i = 0; i < 5; i++) {
    System.out.println(i);
}
```

增强 `for` 常用于遍历集合。

```java
List<String> names = new ArrayList<String>();
names.add("Tom");
names.add("Jerry");

for (String name : names) {
    System.out.println(name);
}
```

---

## 3. 方法

方法必须写返回值类型和参数类型。

```java
public int add(int a, int b) {
    return a + b;
}
```

没有返回值时使用 `void`。

```java
public void printName(String name) {
    System.out.println(name);
}
```

---

## 4. 类和对象

Java 代码通常写在 `class` 中。

```java
public class User {
    private Long id;
    private String name;
    private int age;

    public Long getId() {
        return id;
    }

    public void setId(Long id) {
        this.id = id;
    }

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }

    public int getAge() {
        return age;
    }

    public void setAge(int age) {
        this.age = age;
    }
}
```

创建对象、赋值、取值：

```java
User user = new User();
user.setId(1L);
user.setName("Tom");
user.setAge(18);

String name = user.getName();
```

## 5. List

`List` 用来保存一组有顺序的数据。

```java
List<String> names = new ArrayList<String>();

names.add("Tom");
names.add("Jerry");

String first = names.get(0);
int size = names.size();
```

遍历：

```java
for (String name : names) {
    System.out.println(name);
}
```

---

## 6. Map

`Map` 用来保存键值对。

```java
Map<String, Object> map = new HashMap<String, Object>();

map.put("id", 1L);
map.put("name", "Tom");

Object name = map.get("name");
```

---

## 7. 泛型

`<>` 表示集合中元素的类型。

```java
List<String> names = new ArrayList<String>();
List<User> users = new ArrayList<User>();
Map<String, User> userMap = new HashMap<String, User>();
```

在 Java 8 中，右侧类型可以简写：

```java
List<User> users = new ArrayList<>();
Map<String, User> userMap = new HashMap<>();
```

---

## 8. Lambda

Lambda 是 Java 8 引入的简洁函数写法，常用于集合处理。

排序示例：

```java
users.sort((a, b) -> a.getAge() - b.getAge());
```

如果 Lambda 只有一行，可以省略 `{}` 和 `return`。

```java
user -> user.getName()
```

多行写法：

```java
user -> {
    System.out.println(user.getName());
    return user.getAge();
}
```

---

## 9. 方法引用

方法引用是 Lambda 的简写。

```java
User::getName
```

等价于：

```java
user -> user.getName()
```

常见用法：

```java
List<String> names = users.stream()
        .map(User::getName)
        .collect(Collectors.toList());
```

---

## 10. Stream

`Stream` 用来对集合做链式处理。Java 8 中最后通常使用 `collect(Collectors.toList())` 收集结果。

过滤：

```java
List<User> adults = users.stream()
        .filter(user -> user.getAge() >= 18)
        .collect(Collectors.toList());
```

转换：

```java
List<String> names = users.stream()
        .map(User::getName)
        .collect(Collectors.toList());
```

排序：

```java
List<User> sortedUsers = users.stream()
        .sorted(Comparator.comparing(User::getAge))
        .collect(Collectors.toList());
```

去重：

```java
List<String> uniqueNames = names.stream()
        .distinct()
        .collect(Collectors.toList());
```

查找第一个：

```java
User user = users.stream()
        .filter(u -> u.getId().equals(1L))
        .findFirst()
        .orElse(null);
```

遍历：

```java
users.forEach(user -> System.out.println(user.getName()));
```

## 11. 常用 import

上面的集合和 Stream 示例通常需要这些导入：

import java.util.ArrayList;

import java.util.List;

## 12. 最简记忆版

```java
// 变量：类型 变量名 = 值;
String name = "Tom";
int age = 18;
long id = 1L;
double score = 99.5;
boolean enabled = true;

// if
if (条件) {
    代码;
} else {
    代码;
}

// for
for (int i = 0; i < 次数; i++) {
    代码;
}

// 增强 for
for (元素类型 变量 : 集合) {
    代码;
}

// 方法
public 返回值类型 方法名(参数类型 参数名) {
    return 返回值;
}

// 无返回值方法
public void 方法名(参数类型 参数名) {
    代码;
}

// 类
public class 类名 {
    private 类型 字段名;

    public 类型 get字段名() {
        return 字段名;
    }

    public void set字段名(类型 字段名) {
        this.字段名 = 字段名;
    }
}

// 创建对象
类名 对象名 = new 类名();
对象名.set字段名(值);
类型 变量名 = 对象名.get字段名();

// List
List<元素类型> list = new ArrayList<>();
list.add(值);
元素类型 item = list.get(下标);
int size = list.size();

// Map
Map<键类型, 值类型> map = new HashMap<>();
map.put(键, 值);
值类型 value = map.get(键);

// 泛型
List<String> names = new ArrayList<>();
List<User> users = new ArrayList<>();
Map<String, User> userMap = new HashMap<>();

// Lambda
参数 -> 表达式
参数 -> {
    代码;
    return 返回值;
}

// 方法引用
类名::方法名

// Stream
List<结果类型> result = 集合.stream()
        .filter(元素 -> 条件)
        .map(元素 -> 结果)
        .collect(Collectors.toList());

// 排序
List<User> sortedUsers = users.stream()
        .sorted(Comparator.comparing(User::getAge))
        .collect(Collectors.toList());

// 去重
List<String> uniqueNames = names.stream()
        .distinct()
        .collect(Collectors.toList());

// 查找第一个
User user = users.stream()
        .filter(u -> 条件)
        .findFirst()
        .orElse(null);

// 遍历
集合.forEach(元素 -> 代码);

// 常用 import
import java.util.ArrayList;
import java.util.Comparator;
import java.util.HashMap;
import java.util.List;
import java.util.Map;
import java.util.stream.Collectors;
```
