# Java 8 极简入门

目标：掌握 Java 8 最常用的语法，能读懂并写出简单代码。

---

## 1. 基本类型和变量

Java 变量必须声明类型。

```java
String name = "Tom";
int age = 18;
long id = 1001L;
double score = 99.5;
boolean enabled = true;
```

常用类型：

| 类型 | 含义 | 示例 |
| --- | --- | --- |
| `String` | 字符串 | `"Tom"` |
| `int` | 整数 | `18` |
| `long` / `Long` | 较大的整数 | `1001L` |
| `double` | 小数 | `99.5` |
| `boolean` | 布尔值 | `true` / `false` |

## 2. 条件和循环

```java
if (age >= 18) {
    System.out.println("adult");
} else {
    System.out.println("child");
}
```

```java
for (int i = 0; i < 5; i++) {
    System.out.println(i);
}
```

增强 `for` 常用于遍历集合。

```java
List<String> names = new ArrayList<String>();
names.add("Tom");
names.add("Jerry");

for (String name : names) {
    System.out.println(name);
}
```

## 3. 方法

方法必须写返回值类型和参数类型。

```java
public int add(int a, int b) {
    return a + b;
}
```

没有返回值时使用 `void`。

```java
public void printName(String name) {
    System.out.println(name);
}
```

```java
public class User {

        return id;

    public void setId(Long id) {
        this.id = id;

        this.name = name;

    public int getAge() {
        return age;

    public void setAge(int age) {
        this.age = age;
```

```java
User user = new User();
user.setId(1L);
user.setName("Tom");

String name = user.getName();
```

## 5. 接口

接口（Interface）定义一组方法规范，实现类必须实现这些方法。

定义接口：

```java
public interface Animal {
    void speak();
    String getName();
}
```

实现接口：

```java
public class Dog implements Animal {
    private String name;

    public Dog(String name) {
        this.name = name;
    }

    @Override
    public void speak() {
        System.out.println("Woof!");
    }

    @Override
    public String getName() {
        return name;
    }
}

public class Cat implements Animal {
    private String name;

    public Cat(String name) {
        this.name = name;
    }

    @Override
    public void speak() {
        System.out.println("Meow!");
    }

    @Override
    public String getName() {
        return name;
    }
}
```

`@Override`是注解, 用于给编译器提示这段代码的作用, 让编译器进行自动处理, 这里`@Override`告诉编译器这里重写了方法

使用接口：

```java
Animal dog = new Dog("Buddy");
dog.speak();  // 输出：Woof!

Animal cat = new Cat("Whiskers");
cat.speak();  // 输出：Meow!
```

接口的优点：

- **多态性**：同一个接口可以有多个实现
- **规范**：定义统一的方法契约
- **松耦合**：依赖接口而不是具体实现类

常见用法（配合 Stream）：

```java
List<Animal> animals = new ArrayList<>();
animals.add(new Dog("Buddy"));
animals.add(new Cat("Whiskers"));

animals.forEach(animal -> animal.speak());
```

## 6. List

`List` 用来保存一组有顺序的数据。

```java
List<String> names = new ArrayList<String>();

names.add("Tom");
names.add("Jerry");

String first = names.get(0);
int size = names.size();
```

这里的`List`就是`interface` 定义了列表的统一规范, `ArrayList`实现这个这个`interface`

遍历：

```java
for (String name : names) {
    System.out.println(name);
}
```

```java
Map<String, Object> map = new HashMap<String, Object>();
```

## 8. 泛型

`<>` 表示集合中元素的类型。

```java
List<String> names = new ArrayList<String>();
List<User> users = new ArrayList<User>();
Map<String, User> userMap = new HashMap<String, User>();
```

在 Java 8 中，右侧类型可以简写：

```java
List<User> users = new ArrayList<>();
Map<String, User> userMap = new HashMap<>();
```

## 9. Lambda

Lambda 是 Java 8 引入的简洁函数写法，常用于集合处理。

排序示例：

```java
users.sort((a, b) -> a.getAge() - b.getAge());
```

如果 Lambda 只有一��，可以省略 `{}` 和 `return`。

```java
user -> user.getName()
```

多行写法：

```java
user -> {
    System.out.println(user.getName());
    return user.getAge();
}
```

---

## 10. 方法引用

方法引用是 Lambda 的简写。

```java
User::getName
```

等价于：

```java
user -> user.getName()
```

常见用法：

```java
List<String> names = users.stream()
        .map(User::getName)
        .collect(Collectors.toList());
```

---

## 11. Stream

`Stream` 用来对集合做链式处理。Java 8 中最后通常使用 `collect(Collectors.toList())` 收集结果。

过滤：

```java
List<User> adults = users.stream()
        .filter(user -> user.getAge() >= 18)
        .collect(Collectors.toList());
```

转换：

```java
List<String> names = users.stream()
        .map(User::getName)
        .collect(Collectors.toList());
```

排序：

```java
List<User> sortedUsers = users.stream()
        .sorted(Comparator.comparing(User::getAge))
        .collect(Collectors.toList());
```

去重：

```java
List<String> uniqueNames = names.stream()
        .distinct()
        .collect(Collectors.toList());
```

查找第一个：

```java
User user = users.stream()
        .filter(u -> u.getId().equals(1L))
        .findFirst()
        .orElse(null);
```

遍历：

```java
users.forEach(user -> System.out.println(user.getName()));
```

```java
import java.util.ArrayList;
import java.util.Comparator;
```

```java
        List<String> names = new ArrayList<>();

        map.put("id", id);
        map.put("name", name);

        user.setId(id);

        this.age = age;
```

## 14. Windows 安装环境和 VS Code 运行 Java

### 14.1 安装 Microsoft Build of OpenJDK

Java 代码需要 JDK 才能编译和运行。这里推荐直接安装 Microsoft Build of OpenJDK，它是免费的 OpenJDK 发行版，支持 Windows、macOS 和 Linux。

下载地址： https://adoptium.net/temurin/releases/?version=8

```powershell
java -version
javac -version
where java
```

如果 `java` 和 `javac` 都能找到，说明环境变量已经生效。

### 14.2 验证 Java 是否安装成功

打开新的 PowerShell 或 CMD，执行：

```powershell
java -version
javac -version
```

能看到版本号，就说明安装成功。


### 14.3 安装 VS Code 插件

在 VS Code 插件市场搜索并安装：

```text
Extension Pack for Java
```

### 14.4 用 VS Code 写第一个 Java 文件

新建文件：

```text
Hello.java
```

写入代码：

```java
public class Hello {
    public static void main(String[] args) {
        System.out.println("Hello Java");
    }
}
```

注意：文件名 `Hello.java` 要和 `public class Hello` 的类名一致。

### 14.5 在 VS Code 里运行 Java

打开 `Hello.java` 后，通常可以直接点击代码上方的：

```text
Run
```

也可以右键代码文件，选择：

```text
Run Java
```

输出窗口中看到：

```text
Hello Java
```

说明运行成功。

### 14.6 用命令行编译和运行

进入 `Hello.java` 所在目录：

```powershell
cd D:\workspace\hello-java
```

编译：

```powershell
javac Hello.java
```

编译成功后，会生成：

```text
Hello.class
```

运行：

```powershell
java Hello
```

注意：运行时写 `java Hello`，不要写 `java Hello.class`。

完整流程：

```powershell
javac Hello.java
java Hello
```
