# Spring Boot 接口极简入门

目标：掌握 Spring Boot 最常用的接口写法，能启动项目、写 GET / POST 接口、接收前端参数，并按 `controller -> service` 做简单分层。

---

## 1. 创建 Spring Boot 项目

### 1.1 安装 SDKMAN

先安装 SDKMAN（Java 生态的版本管理工具）：

```bash
curl -s "https://get.sdkman.io" | bash
```

安装完成后，重启终端或执行：

```bash
source "$HOME/.sdkman/bin/sdkman-init.sh"
```

验证安装：

```bash
sdk version
```

### 1.2 安装 Spring Boot CLI

```bash
sdk install springboot
```

验证安装：

```bash
spring --version
```

其他安装方式（备选）：

* **Mac**：`brew install springboot`
* **Windows**：下载安装包 <https://repo.spring.io/release/org/springframework/boot/spring-boot-cli/>

### 1.3 使用 spring init 创建项目

一行命令生成 Maven 项目：

```bash
spring init \
  --dependencies=web \
  --groupId=com.example \
  --artifactId=demo \
  --package-name=com.example.demo \
  --java-version=17 \
  demo
```

进入项目目录：

```bash
cd demo
```

生成的项目结构：

```text
demo/
├── src/
│   └── main/
│       ├── java/
│       │   └── com/example/demo/
│       │       └── DemoApplication.java
│       └── resources/
│           └── application.properties
├── mvnw
├── mvnw.cmd
└── pom.xml
```

### 1.3 关键配置说明

生成的 `pom.xml` 已包含核心配置：

| 配置项 | 说明 |
| --- | --- |
| `spring-boot-starter-parent` | Spring Boot 父工程，统一管理依赖版本 |
| `spring-boot-starter-web` | Web 开发核心依赖（含 Spring MVC、内嵌 Tomcat） |
| `spring-boot-maven-plugin` | Spring Boot Maven 插件，支持打包和运行 |
| `java.version` | Java 版本，建议 8 或 17，按本地环境选择 |

## 2. 项目目录结构

常见结构如下：

```text
src/main/java/com/example/demo
├── DemoApplication.java
├── controller
│   └── HelloController.java
├── service
│   └── UserService.java
└── model
    └── User.java
```

先记住：

* `controller`：接收前端请求，返回数据
* `service`：写业务逻辑
* `model`：定义数据对象

## 3. 启动项目

项目已自带入口类 `DemoApplication.java`：

```java
package com.example.demo;

import org.springframework.boot.SpringApplication;
import org.springframework.boot.autoconfigure.SpringBootApplication;

@SpringBootApplication
public class DemoApplication {

    public static void main(String[] args) {
        SpringApplication.run(DemoApplication.class, args);
    }
}
```

在项目根目录执行启动命令：

```bash
mvn spring-boot:run
```

或使用项目自带的 Maven Wrapper（无需安装 Maven）：

```bash
# Linux/Mac
./mvnw spring-boot:run

# Windows
mvnw.cmd spring-boot:run
```

启动后，默认访问地址：

```text
http://localhost:8080
```

## 4. 写第一个 GET 接口

创建 `controller/HelloController.java`：

```java
package com.example.demo.controller;

import java.util.HashMap;
import java.util.Map;

import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.RestController;

@RestController
public class HelloController {

    @GetMapping("/hello")
    public Map<String, Object> hello() {
        Map<String, Object> result = new HashMap<>();
        result.put("msg", "hello world");
        return result;
    }
}
```

访问：

```text
GET http://localhost:8080/hello
```

返回：

```json
{
  "msg": "hello world"
}
```

## 5. 必学注解

### `@RestController`

表示这个类专门用来写接口。

返回 `String`、对象、`Map`、`List` 时，Spring Boot 会自动转成响应内容。

```java
@RestController
public class HelloController {
}
```

### `@GetMapping`

表示接收 GET 请求，常用于查询数据。

```java
@GetMapping("/hello")
public String hello() {
    return "hello";
}
```

### `@PostMapping`

表示接收 POST 请求，常用于新增数据或提交表单。

```java
@PostMapping("/user")
public User createUser(@RequestBody User user) {
    return user;
}
```

### `@RequestParam`

接收 Query 参数。

```java
@GetMapping("/user")
public String getUser(@RequestParam Long id) {
    return "user id = " + id;
}
```

请求：

```text
GET http://localhost:8080/user?id=1
```

### `@RequestBody`

接收 JSON 请求体。

```java
@PostMapping("/user")
public User createUser(@RequestBody User user) {
    return user;
}
```

请求体：

```json
{
  "id": 1,
  "name": "Tom",
  "age": 18
}
```

## 6. 定义 User 对象

创建 `model/User.java`：

```java
package com.example.demo.model;

public class User {

    private Long id;
    private String name;
    private Integer age;

    public Long getId() {
        return id;
    }

    public void setId(Long id) {
        this.id = id;
    }

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }

    public Integer getAge() {
        return age;
    }

    public void setAge(Integer age) {
        this.age = age;
    }
}
```

## 7. GET 接收 Query 参数

创建 `controller/UserController.java`：

```java
package com.example.demo.controller;

import com.example.demo.model.User;
import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.RequestParam;
import org.springframework.web.bind.annotation.RestController;

@RestController
public class UserController {

    @GetMapping("/user")
    public User getUser(@RequestParam Long id) {
        User user = new User();
        user.setId(id);
        user.setName("Tom");
        user.setAge(18);
        return user;
    }
}
```

访问：

```text
GET http://localhost:8080/user?id=1
```

返回：

```json
{
  "id": 1,
  "name": "Tom",
  "age": 18
}
```

## 8. POST 接收 JSON

在 `UserController` 中增加：

```java
import org.springframework.web.bind.annotation.PostMapping;
import org.springframework.web.bind.annotation.RequestBody;
```

```java
@PostMapping("/user")
public User createUser(@RequestBody User user) {
    return user;
}
```

请求：

```text
POST http://localhost:8080/user
Content-Type: application/json
```

请求体：

```json
{
  "id": 2,
  "name": "Jerry",
  "age": 20
}
```

返回：

```json
{
  "id": 2,
  "name": "Jerry",
  "age": 20
}
```

## 9. 简单分层：controller -> service

不要把所有逻辑都写在 `controller` 里。

先记住这个结构：

```text
前端请求 -> controller -> service -> 返回结果
```

创建 `service/UserService.java`：

```java
package com.example.demo.service;

import com.example.demo.model.User;
import org.springframework.stereotype.Service;

@Service
public class UserService {

    public User getUser(Long id) {
        User user = new User();
        user.setId(id);
        user.setName("Tom");
        user.setAge(18);
        return user;
    }

    public User createUser(User user) {
        return user;
    }
}
```

修改 `UserController.java`：

```java
package com.example.demo.controller;

import com.example.demo.model.User;
import com.example.demo.service.UserService;
import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.PostMapping;
import org.springframework.web.bind.annotation.RequestBody;
import org.springframework.web.bind.annotation.RequestParam;
import org.springframework.web.bind.annotation.RestController;

@RestController
public class UserController {

    private final UserService userService;

    public UserController(UserService userService) {
        this.userService = userService;
    }

    @GetMapping("/user")
    public User getUser(@RequestParam Long id) {
        return userService.getUser(id);
    }

    @PostMapping("/user")
    public User createUser(@RequestBody User user) {
        return userService.createUser(user);
    }
}
```

## 10. 当前阶段只需要记住

| 注解 | 作用 |
| --- | --- |
| `@RestController` | 声明接口类 |
| `@GetMapping` | 写 GET 接口 |
| `@PostMapping` | 写 POST 接口 |
| `@RequestParam` | 接收 Query 参数 |
| `@RequestBody` | 接收 JSON 请求体 |
| `@Service` | 声明业务类 |
